Description for HTML tags

A list of Description for 5 html tags

  1. <h1> - This is an opening for header tag and is used for giving heading to html document. It closes with </h1>.
  2. <p> - This is paragraph tag and is used for writing paragraph in html document. It closes with </p>.
  3. <br> - This tag is used to to break a line and anything after this tag continues from next line.
  4. <ol> - This tag is used to keep an ordered list. It closes with </ul>.
  5. <head> - This tag defines the header of an html document. It closes with </header>.
